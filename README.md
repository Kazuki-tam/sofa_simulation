# simulatuin_app

## プロジェクトのセットアップ
```
yarn install
```

### 開発時のコンパイルとリロード（ローカルサーバー立ち上げ）
```
yarn run serve
yarn run serve --open
```

### コンパイルと圧縮
```
yarn run build
```

### テスト
```
yarn run test
```

### ファイルのLint
```
yarn run lint
```
